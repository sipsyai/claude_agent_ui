# Strapi Project Helper Skill

Bu skill, Strapi 5 projenizde en cok kullanilan konularda detayli dokumantasyon ve hazir kod ornekleri saglar.

## Ozellikler

- Read-only: Sadece dokumanlara bakip bilgi verir, kod yazmaz
- Kapsamli: 8 ana konu alani
- Orneklerle: Hazir kod ornekleri ve gercek proje sablonlari
- Pratik: Gercek dunya senaryolari
- Guncel: Strapi 5 icin hazirlanmistir

## Kapsanan Konular

### 1. Content-Type Builder & Best Practices (YENI!)
- Field type secimi rehberi
- Isimlendirme standartlari
- Validation kurallari
- Performans optimizasyonu
- Guvenlik best practices
- Yaygin hatalar ve cozumleri

### 2. Collection Types (YENI!)
- Collection Type nedir ve ne zaman kullanilir
- Olusturma ve yapilandirma
- Tum field tipleri (Text, Number, Date, Media, JSON, UID)
- CRUD islemleri ve API ornekleri
- Lifecycle hooks
- Draft & Publish mode
- Gercek proje ornekleri (Blog, E-commerce, Event sistemi)

### 3. Single Types (YENI!)
- Single Type nedir ve ne zaman kullanilir
- Collection Type ile farklar
- Kullanim alanlari (Homepage, Settings, About, Contact)
- API endpoint farklari
- Component kullanimi
- Best practices

### 4. Components (YENI!)
- Component nedir ve avantajlari
- Shared vs Non-Shared components detayli karsilastirma
- Repeatable components
- Dynamic Zones ile sayfa olusturma
- Nested components (component icinde component)
- Hazir component kutuphanesi (SEO, Address, Button, Hero, vb.)
- Frontend rendering ornekleri

### 5. Content-Type Relations
- One-to-Many, Many-to-Many iliskiler
- Populate kullanimi ve best practices
- Deep relation queries
- Iliski yonetimi

### 6. Custom Controllers & Services
- Custom endpoint olusturma
- Business logic yazma
- Service katmani kullanimi
- Error handling ve validation

### 7. Database Queries & Filters
- Gelismis filtreleme (eq, ne, lt, gt, contains, etc.)
- Tarih araligi sorgulari
- Kompleks AND/OR filtreleri
- Dynamic filter builder

### 8. File Upload & Media
- Tek ve coklu dosya yukleme
- URL'den dosya yukleme
- Base64 upload
- React ornekleri

## Yeni Eklenen Konular

- Content-Type Builder Best Practices dokumantasyonu
- Collection Types detayli rehberi
- Single Types kapsamli dokumantasyonu
- Components tam rehberi
- Blog sistemi tam ornegi
- E-commerce sistemi tam ornegi
- Hazir component kutuphanesi
