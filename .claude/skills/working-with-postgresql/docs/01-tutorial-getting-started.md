# PostgreSQL 18 Tutorial - Part I

## Overview

The PostgreSQL Tutorial provides "an introduction to PostgreSQL, relational database concepts, and the SQL language." No prior Unix or programming experience is required, though basic computer proficiency is assumed.

Upon completing this tutorial, readers should progress to Part II for deeper SQL understanding, Part IV for application development guidance, or Part III for installation management topics.
