# TypeScript in Strapi 5

## Introduction

TypeScript is a superset of JavaScript that adds static typing capabilities. According to the Strapi documentation, "TypeScript adds an additional type system layer above JavaScript, which means that any valid JavaScript code is also valid TypeScript code."

In Strapi development, TypeScript enables:
- A more type-safe codebase
- Automatic type generation
- Enhanced autocompletion features

**Note:** This feature requires Strapi version 4.3.0 or later.

## Getting Started with TypeScript in Strapi

### Option 1: Create a New Project

Start a fresh Strapi project with TypeScript support:

**Using Yarn:**
```bash
yarn create strapi-app my-project --typescript
```

**Using NPM:**
```bash
npx create-strapi-app@latest my-project --typescript
```

### Option 2: Add to Existing Project

TypeScript support can be added to an already-established Strapi project following the conversion documentation.
