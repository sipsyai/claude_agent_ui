# Flows sayfasının react flow ile sürükle bırak ve aralarında ilişki veri transferleri yapan n8n , langchain benzer bir yapıda olması

Flows sayfasının react flow ile sürükle bırak ve aralarında ilişki veri transferleri yapan n8n , langchain benzer bir yapıda olması Bu özellik sayesinde, agentler, süreçler, tetiklemeler vs. yani mevcut yapıyı aslında React Flow'a taşısam, burada yeter. 

React Flow veya daha uygun bir teknoloji varsa bilmiyorum; ama React Flow ile daha kolay yapılır gibi. Daha uygun teknoloji varsa onu da araştırın, onu da kullanabiliriz

## Rationale
Burada yapacağımız sürükle bırakla birbirine bağlamakla, aslında daha görsel, daha kolay tasarlanabilecek paralel yapılar kurgulayabilecek, daha user-friendly bir yapıya sahip olacak. O yüzden aslında React Flow'a geçiyoruz
